权限修饰符

概述

  • public ——公共的
  • protected——受保护的
  • default——默认的
  • private——私有的

区别

public protected default private
同一个类中 Yes Yes Yes Yes
同一个包中 Yes Yes Yes
不同包的继承类 Yes Yes
不同包的无关类 Yes

【空】表示不能访问

权限使用建议

  • 成员变量使用private，隐藏细节
  • 成员方法使用public，方便调用方法
  • 构造方法使用public ，方便创建对象
